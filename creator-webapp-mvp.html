Feature list · MVP

Brand-led invite link
Tokenized URL routed via WhatsApp / email. Carries brand + offer context.
Phyllo OAuth claim
Identity verification + analytics data pipe. IG / YT / TikTok / X.
Pipeline dashboard
Viewed / shortlisted / active / pending stats + brand-viewing strip.
Offer inbox
Filterable list with status badges (new, viewed, expiring, accepted).
Offer detail + accept
Brief, rate, timeline, deliverables. Accept triggers take-rate invoice.
Counter offer
Rate / deliverables editable. One-shot back-and-forth with brand.
Active deal view
Progress stepper (brief → drafts → approved → posted → paid).
Deliverables upload
Per-deliverable file upload + inline brand checklist.
Revision feedback
Versioned drafts, scoped notes, deadline for revision.
WhatsApp pipe
Composio bridges creator's WhatsApp ↔ brand's in-app inbox.
Profile + rate card
Rates, niches, languages, blocked brands, payment details (UPI).
Connected accounts
Add YouTube etc. post-claim to unlock cross-platform analytics.
Cross-platform insights
Unified reach, engagement, growth across IG / YT / TikTok via Phyllo.
Brand-deal performance
Paid posts vs. your organic baseline · per-deal & per-category insights.
Audience demographics
Age, gender, locations, interests — used for rate negotiations.
